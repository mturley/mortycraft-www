Configuration Details and Plugins

Currently, this server is running on a Linode 1024 ($20/mo) Ubuntu server owned and operated by /u/yooman (Mike Turley). It is a bukkit server with a few minimal mods for control and permissions, otherwise running vanilla Minecraft. We are running:

  • CraftBukkit 1.7.5-R0.1 (yes, running a dev version to keep up with Mojang's Minecraft client. We'll stay on top of version updates)
  • Essentials 2.13.1
  • WorldEdit 5.6 (if you'd like to join the building effort, ask us for access permissions to WorldEdit and WorldGuard)
  • WorldGuard 5.9
  • PermissionsBukkit 2.2
  • DynMap 1.9.3

And so far that's all. We have plans to add multiverse, but no further modding (nothing crazy). The goal is for mostly vanilla gameplay.

How do I play?

If you're new to multiplayer minecraft servers, here's how to play:

  1. Click "Multiplayer" from the Minecraft Main Menu
  2. Click "Add Server"
  3. Enter the address mortycraft.net (or if that doesn't work, 96.126.111.207) for Server Address
  4. Click "Done"
  5. Select the server and click "Join Server"
