Server Rules: ANARCHY

Currently there are are two very simple rules:

  1. No client-side game mods or hacks are allowed (non-xraying themes and skins are an exception). Play using only the official Minecraft client available from Mojang.
  2. Anarchy rules are in effect. PVP, raiding, stealing, griefing, etc are all allowed.

Go nuts, but please help us preserve the server's status as a fun place for everyone else to go nuts too.

For details and discussion, see our subreddit at /r/rickandmortyserver.

  Mumble Voice Chat (Disabled)

In addition to Minecraft, we are also running a Mumble server here on mortycraft.net. To join us on voice chat:

Since switching to FragNet, we have not yet replaced our Mumble server (it costs extra at FragNet).

  1. Download, install and configure a Mumble client for your operating system. See "Get Mumble" on Sourceforge.
  2. Open the server list and click "Add New..."
  3. Label the server whatever you like, and enter mortycraft.net for the address and 64738 for the port.
  4. Click OK, then click Connect. Introduce yourself and enjoy!

  How do I play?

If you're new to multiplayer minecraft servers, here's how to play:

  1. Click "Multiplayer" from the Minecraft Main Menu
  2. Click "Add Server"
  3. Enter the address mortycraft.net (or if that doesn't work, 96.126.111.207) for Server Address
  4. Click "Done"
  5. Select the server and click "Join Server"

See you in game!

  Configuration Details and Plugins

Currently, this server is running on a FragNet Minecraft Bronze Package ($30/mo for 3GB RAM) owned and operated by /u/yooman (Mike Turley).
It is a bukkit server with a few minimal mods for control and permissions, otherwise running vanilla Minecraft. We are running:

  • CraftBukkit 1.7.8-R0.1 (yes, running a dev version to keep up with Mojang's Minecraft client. We'll stay on top of version updates)
  • BukkitHTTPD 0.2.5 (to host this webpage!)
  • Essentials 2.13.1
  • WorldEdit 5.6 (if you'd like to join the building effort, ask us for access permissions to WorldEdit and WorldGuard)
  • WorldGuard 5.9
  • PermissionsBukkit 2.2
  • Vault 1.2.31 (dependency APIs for Enjin)
  • Enjin DonationCraft 2.5.6 (I'll be setting this up soon to facilitate reserved player slots for people who donate)

And so far that's all. We have plans to add factions and multiverse, but no further modding (nothing crazy). The goal is for mostly vanilla gameplay.
